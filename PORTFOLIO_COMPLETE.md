## 🔧 Customization Guide

### Change Colors
File: `/app/globals.css`
```css
:root {
  --primary: oklch(0.25 0.08 250);    /* Change navy to your color */
  --accent: oklch(0.3 0.12 25);       /* Change gold to your color */
}
```

File: `/components/header.tsx`
- Links point to correct pages
- Mobile menu responsive
- Logo customizable
